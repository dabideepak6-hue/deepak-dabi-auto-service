What is an EV Charger?

EV Charger converts AC power from electricity supply into controlled DC power to charge the vehicle battery.

  • ✔ AC Input Protection
  • ✔ AC to DC Conversion
  • ✔ Charging Control
  • ✔ Battery Communication
  • ✔ Safety Protection

EV Charger Working Process

1. AC Input

230V AC supply enters charger through protection circuit.

2. Rectifier

Converts AC voltage into DC voltage.

3. PFC Circuit

Improves power efficiency and reduces harmonic losses.

4. SMPS Section

Controls high frequency switching and voltage conversion.

5. Output Section

Provides regulated DC output to battery.

6. Communication

Communicates with BMS for safe charging.

Charger Main Components

Fuse

Protects charger from input short circuit.

MOSFET

Controls high frequency switching operation.

Transformer

Provides voltage isolation and conversion.

Capacitor

Filters DC voltage ripple.

Cooling Fan

Controls charger temperature.

Control IC

Manages charging operation.

Common Charger Faults

Charger Not ON

Possible Causes:
• Input Fuse Open
• AC Supply Missing
• PCB Fault
• Control IC Damage

Charging Stops Suddenly

Possible Causes:
• Over Temperature
• Fan Not Working
• BMS Protection

Low Charging Current

Possible Causes:
• Weak Components
• Output Stage Problem
• Communication Error

Charger Heating

Possible Causes:
• Fan Failure
• MOSFET Heating
• Over Load

Charger Testing Procedure

  1. Disconnect Charger From Vehicle
  2. Check AC Input Supply
  3. Check Fuse Continuity
  4. Check PCB Components
  5. Check Capacitor Condition
  6. Check MOSFET Short Test
  7. Measure Output Voltage
  8. Check Charging Communication
  9. Perform Charging Test
